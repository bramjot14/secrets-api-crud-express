# Secrets API CRUD Project

This project is a simple **Node.js and Express application** that interacts with the **Secrets API** to perform CRUD operations using **Axios**.

## Features
- Get a secret by ID
- Create a new secret
- Update a secret using PUT
- Partially update a secret using PATCH
- Delete a secret

## Technologies
- Node.js
- Express.js
- Axios
- EJS

## Installation
Install dependencies:
npm install

## Run the Project
node index.js

Open in browser:
http://localhost:3000
